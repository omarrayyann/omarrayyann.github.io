Connected Components in an Undirected Graph

Graphs

C++ Code

The algorithm counts the number of connected components in an undirected graph by performing a DFS (Depth-First Search) traversal from each unvisited vertex. It keeps track of the visited vertices to avoid revisiting them, and increments a counter each time it completes a DFS traversal from an unvisited vertex.

O(m+n), where m is the number of edges and n is the number of vertices 

            vector <bool> visited;
            int number_of_connected_components(vector <vector<int>glt graph) {

            visited.resize(graph.size(), false);
            int components = 0;

            for (int i = 0; i<graph.size(); i++) {
              if (!visited.at(i)) {
                DFS(graph, i);
                components++;
              }
            }
            
            return components;
            }
